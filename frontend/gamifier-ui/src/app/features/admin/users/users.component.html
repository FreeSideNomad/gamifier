CSV File Format

Your CSV file should contain the following columns:

  • name - Full name of the user
  • email - Email address (must be unique)
  • rank - User rank (CADET, ENSIGN, etc.)
  • department - Department name
  • role - USER or ADMIN
  • managerId - (Optional) Email of the user's manager

Important Notes

  • All imported users will receive temporary passwords
  • Users will be required to change their password on first login
  • Invalid entries will be skipped and reported
  • Maximum file size: 10MB
